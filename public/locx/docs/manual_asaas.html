Conectar Asaas / PIX

  1. Crie ou acesse sua conta Asaas e conclua a validacao cadastral.
  2. No painel Asaas, gere a API Key do ambiente desejado.
  3. No LocX, abra Asaas, escolha o ambiente e cole a API Key.
  4. Configure a URL de webhook no painel Asaas e use o mesmo token exibido no LocX.
  5. Em Gateway PIX principal, escolha Asaas.
  6. Cadastre um cliente com CPF/CNPJ valido, crie uma cobranca e use Gerar PIX.

Webhook

Use este formato, trocando pelo seu dominio:

https://seusite.com/webhooks/asaas

Importante: localhost nao recebe notificacoes do Asaas. O endereco precisa ser publico e usar HTTPS.

Producao

Depois que o sandbox funcionar, troque o ambiente para producao e use a API Key de producao. Faca primeiro um PIX de valor baixo.
