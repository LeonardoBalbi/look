Conectar WhatsApp Cloud API

  1. No Meta for Developers, crie ou abra um aplicativo e adicione o produto WhatsApp.
  2. Copie o WABA ID (ID da conta do WhatsApp Business) e o Phone Number ID.
  3. Crie um usuário do sistema no Business Manager e gere um Access Token permanente. Tokens temporários expiram e interrompem os envios.
  4. No LocX, abra WhatsApp API, escolha Oficial, preencha os dados e salve.
  5. Clique em Validar na Meta. O teste também confirma se o template e o idioma estão aprovados.
  6. Na Meta, configure a URL pública do webhook e use o mesmo Verify Token exibido no LocX.

Template de cobrança

Crie e aprove na Meta o template locx_cobranca_atraso. Informe no LocX exatamente o mesmo nome e idioma, por exemplo pt_BR, com cinco variáveis nomeadas no corpo:

  1. Nome do cliente
  2. Placa da motocicleta
  3. Dias em atraso
  4. Saldo atualizado
  5. PIX copia e cola

Importante: a aprovação da empresa na Meta não renova o Access Token e não aprova automaticamente cada idioma do template.

Webhook

Use este formato, trocando pelo seu domínio:

https://seusite.com/webhooks/whatsapp

Importante: a Meta não consegue acessar localhost. O endereço precisa ser público e usar HTTPS.
