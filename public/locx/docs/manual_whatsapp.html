Conectar WhatsApp Cloud API

  1. No Meta for Developers, crie ou abra um aplicativo e adicione o produto WhatsApp.
  2. Copie o Phone Number ID e gere um Access Token permanente.
  3. No LocX, abra WhatsApp API, escolha Oficial, cole os dois dados e salve.
  4. Clique em Validar credenciais.
  5. Na Meta, configure a URL pública do webhook e use o mesmo Verify Token exibido no LocX.

Template de cobrança

Crie e aprove na Meta o template locx_cobranca_atraso, em português, com cinco variáveis no corpo:

  1. Nome do cliente
  2. Placa da motocicleta
  3. Dias em atraso
  4. Saldo atualizado
  5. PIX copia e cola

Webhook

Use este formato, trocando pelo seu domínio:

https://seusite.com/locx/webhooks/whatsapp.php

Importante: a Meta não consegue acessar localhost. O endereço precisa ser público e usar HTTPS.
