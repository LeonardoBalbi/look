Conectar PagBank / PIX

  1. Entre na área de desenvolvedores do PagBank e obtenha o Access Token do ambiente desejado.
  2. No LocX, abra PagBank, escolha api oficial e comece pelo ambiente sandbox.
  3. Cole o Access Token e informe a URL pública do webhook.
  4. Clique em Testar conexão.
  5. Cadastre um cliente com CPF e e-mail válidos, crie uma cobrança e use Gerar PIX.

Webhook

Use este formato, trocando pelo seu domínio:

https://seusite.com/webhooks/pagbank

Importante: localhost não recebe notificações do PagBank. O endereço precisa ser público e usar HTTPS.

Produção

Depois que o sandbox funcionar, troque o ambiente para produção e use o token de produção. Faça primeiro um PIX de valor baixo.
